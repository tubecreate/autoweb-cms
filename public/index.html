Quản lý API Key & Tích hợp WordPress

Website: -

Tạo và quản lý các khóa API để kết nối với WordPress REST API của website.

🔒

Khóa API đã được tạo thành công!

Sao chép và lưu trữ an toàn. Khóa này sẽ không bao giờ hiển thị lại.

Khóa API hiện có
Tên mô tả Tài khoản Key Prefix Ngày tạo Thao tác

Tích hợp WordPress REST API với website của bạn.

Chọn ngôn ngữ:
cURL Scripts
# Đăng bài viết mới
curl -X POST "https://[SITE_URL]/wp-json/wp/v2/posts" \
  -u "[USERNAME]:[API_KEY]" \
  -H "Content-Type: application/json" \
  -d '{"title":"Tiêu đề","content":"Nội dung","status":"publish"}'
JavaScript Fetch
const token = btoa(`${username}:${apiKey}`);
fetch('https://[SITE_URL]/wp-json/wp/v2/posts', {
  method: 'POST',
  headers: { 'Authorization': `Basic ${token}`, 'Content-Type': 'application/json' },
  body: JSON.stringify({ title: 'Tiêu đề', content: 'Nội dung', status: 'publish' })
});
Python Requests
import requests
response = requests.post(
    'https://[SITE_URL]/wp-json/wp/v2/posts',
    json={"title": "Tiêu đề", "content": "Nội dung", "status": "publish"},
    auth=('[USERNAME]', '[API_KEY]')
)
